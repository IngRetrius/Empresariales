Sistema de Gestión Agropecuaria

Universidad de Ibagué - Tolima, Colombia

Sistema web profesional para la gestión integral de productos agrícolas y sus cosechas. Desarrollado como parte del curso de Desarrollo de Aplicaciones Empresariales, este sistema implementa una arquitectura distribuida cliente-servidor con servicios REST.

Versión

2.0.0

Curso

Aplicaciones Empresariales

Año

Estudiante

Funcionalidades Principales

  • Gestión completa de productos agrícolas (CRUD)
  • Registro y control de cosechas asociadas a productos
  • Relación maestro-detalle entre productos y cosechas
  • Búsquedas individuales y filtros avanzados
  • Estadísticas y reportes del sistema
  • Interoperabilidad entre múltiples clientes

Tecnologías Utilizadas

HTML5 CSS3 JavaScript ES6+ REST API Java Spring Boot
